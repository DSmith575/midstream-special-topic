# special-topic

## Installation & Setup

### FFMPEG
From the CMD  
May require elevated permissions to install
```
choco install ffmpeg
```

### Virtual Environment
From the CMD
```
py -m venv .venv
```

Accessing the environment
```
.venv\Scripts\activate
```

### Install packages
```
pip install -r requirements.txt
```

### Running the app locally
```
py main.py
```

Open the app in any browser
```
http://127.0.0.1:5000/
```
